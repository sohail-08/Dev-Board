Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. They are:
1.By ID: getElementById();
2.By Class: getElementsByClassName();
3.By Tag: getElementsByTagName();
4.Using selectors: querySelector(); and querySelectorAll();
5.By name: getElementsByName();

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML content of an element as a string. It includes html tags and elements.
On the other hand innerText and textContent gets or sets the text inside an element. But textContent includes hidden files and innerText doesn't

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where we add a single event listener to a parent element instead of adding multiple event listeners to individual child elements

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event starts at the target element and then bubbles up through its ancestors, triggering event handlers on each parent element along the way

Question-5: How do you create, add, and remove elements using JavaScript?

Creating elements:We can create a new element using the document.createElement() method
Adding elements:We can add a new element using the appendChild(), append(), prepend(), insertBefore() or innerHTML method
Removing elements:We can remove an element using the remove(), remove() or innerHTML="" method
